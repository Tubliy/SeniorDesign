Project Overview

This senior design project demonstrates a glove-based interface for controlling a drone using inductive coil sensors and machine learning. By mapping coil signals to discrete hand gestures, the glove allows the user to issue high-level control commands (e.g., take off, land, move, rotate) without a traditional controller.

Our goals are to explore intuitive human–drone interaction, evaluate the performance of gesture recognition from inductive sensor data, and design a complete embedded system from sensing to actuation.

Hardware Design

  • Sensing: Inductive coils embedded in a glove to measure finger proximity/position.
  • Processing: On-board microcontroller to read coil values and preprocess signals.
  • Communication: Wireless link to the drone (e.g., Wi-Fi/Bluetooth/custom RC interface).
  • Power: Battery-powered wearable design for fully untethered control.

Machine Learning Pipeline

The glove streams raw inductive coil readings, which are converted into features and fed into a trained classifier that predicts the current gesture.

  1. Collect labeled data for each gesture using the glove.
  2. Preprocess signals (filtering, normalization, feature extraction).
  3. Train a machine learning model to map coil features to gesture labels.
  4. Deploy the trained model on the glove’s embedded processor or a companion device.
  5. Send the corresponding control command to the drone in real time.
